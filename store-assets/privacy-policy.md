# Privacy Policy

**Visual GitHub Reporter by Studio N.O.P.E.**

Last updated: March 23, 2026

## Overview

Visual GitHub Reporter is a browser extension that helps you report visual issues to GitHub. We take your privacy seriously and designed this extension to minimize data collection.

## Data Collection

### What we collect
**Nothing.** This extension does not collect, store, or transmit any data to Studio N.O.P.E. or any third party.

### What stays on your device
- Your GitHub Personal Access Token (stored in Chrome's sync storage)
- Your list of configured repositories
- Your extension preferences

### What goes to GitHub
When you create an issue, the following is sent directly to GitHub's API:
- Screenshot of the selected area
- Issue title and description you provide
- Page URL
- Browser and environment metadata

This data is sent directly from your browser to GitHub. We do not operate any servers that process your data.

## Permissions

This extension requires the following permissions:

- **activeTab**: To capture screenshots of the current tab
- **storage**: To save your settings and GitHub token locally
- **Host permissions (all URLs)**: To work on any website you want to report issues from

## Third-Party Services

This extension communicates only with:
- **GitHub API** (api.github.com): To create issues and upload screenshots

## Data Security

- Your GitHub token is stored in Chrome's encrypted sync storage
- All communication with GitHub uses HTTPS
- No data is ever sent to Studio N.O.P.E. servers

## Children's Privacy

This extension is not directed at children under 13 and does not knowingly collect information from children.

We may update this privacy policy from time to time. Changes will be posted in the extension's store listing.
