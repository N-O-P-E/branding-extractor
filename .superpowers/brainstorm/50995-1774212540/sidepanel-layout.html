Key Design Decisions

  • Instrument Serif for all headings — elegant, editorial feel
  • DM Sans for everything else — clean, readable

  • Side Panel transitions: Home → Create Issue (with back button)
  • Labels & Assignees always visible as optional dropdowns
  • Purple (#8b5cf6) as the accent throughout
